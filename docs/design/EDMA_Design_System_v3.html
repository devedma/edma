Design principles

Clarity first
Every screen answers one question. Data density without visual overload.
Trust through transparency
On-chain data one click away. Every number links to its source transaction.
Progressive disclosure
Summary first, detail on demand. Metrics up top, raw data below.
Consistent language
Same component, same interaction, everywhere. Modular across all EDMA apps.
