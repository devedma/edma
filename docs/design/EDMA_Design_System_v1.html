Design principles

Clarity first
Every screen answers one question. Data density without visual overload. White space is a feature, not waste.
Trust through transparency
On-chain data is always one click away. Every number links to its source transaction. Verifiable by default.
Progressive disclosure
Summary first, detail on demand. Metrics at the top, raw data below. Never overwhelm a new user.
Consistent language
Same component, same interaction, everywhere. Modular and composable across explorer, bridge, deal room.
