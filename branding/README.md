# EDMA Branding Assets

Public hosting for branding assets used in places that need a stable
public URL (email signatures, partner sites, etc).

## Files

- `edma-logo-signature.png` — 112×112 PNG of the EDMA mark with
  transparent background. Used in the email signature for vio@edma.co
  (displayed at 56×56 on retina screens).

## CDN URLs

Serve via jsDelivr for fast, cached delivery:

```
https://cdn.jsdelivr.net/gh/devedma/edma@main/branding/edma-logo-signature.png
```

Or directly from GitHub raw (slower, but works):

```
https://raw.githubusercontent.com/devedma/edma/main/branding/edma-logo-signature.png
```
